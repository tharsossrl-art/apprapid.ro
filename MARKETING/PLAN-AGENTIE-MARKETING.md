## 1. CONCEPTUL DE BUSINESS

### Viziunea
Doua firme interconectate care se alimenteaza reciproc:

```
┌─────────────────────┐          ┌─────────────────────┐
│   AGENTIA MARKETING │◄────────►│    APPRAPID.RO      │
│   (servicii lunare) │          │ (produse web/app)   │
│                     │          │                     │
│  Social Media       │  cross   │  Site-uri PWA       │
│  Ads Management     │◄─sell───►│  Aplicatii web      │
│  Content Creation   │          │  Landing pages      │
│  SEO                │          │  E-commerce         │
│  Branding           │          │  Mentenanta         │
│  Video Production   │          │                     │
│  Email Marketing    │          │  249 - 899 EUR      │
│  Consultanta        │          │  (one-time)         │
│                     │          │                     │
│  300 - 2500 EUR/luna│          │  + 25 EUR/luna      │
│  (recurent)         │          │  (hosting)          │
└─────────────────────┘          └─────────────────────┘
```

### Cum se leaga cele doua firme

1. **Agentia -> AppRapid:** Fiecare client de marketing are nevoie de site/app. Agentia recomanda AppRapid pentru dezvoltare. Comision intern sau pret preferential.

2. **AppRapid -> Agentia:** Fiecare client care isi face site/app are nevoie de marketing. AppRapid recomanda agentia pentru promovare.

3. **Pachete combinate:** Oferte bundle "Site + Marketing" la pret redus = imbatabil pe piata.

4. **Retentie dubla:** Clientul plateste lunar agentia (marketing) + lunar AppRapid (hosting). Doua surse de venit recurent de la acelasi client.

### De ce merge

- **Piata din Romania:** 17.8 milioane utilizatori internet, 12.9 milioane pe social media
- **E-commerce Romania:** 7.7 miliarde EUR in 2024, creste 8%/an
- **74% din tranzactii** se fac pe mobil = PWA-urile AppRapid sunt perfecte
- **Putine agentii** combina dezvoltare + marketing sub acelasi umbrela
- **AI-First approach** = costuri mai mici, marje mai mari, rezultate mai rapide

## 3. DIFERENTIEREA NOASTRA - DE CE SA NE ALEAGA PE NOI

### USP-uri principale

1. **AI-First Agency** - Folosim AI pentru tot (video, foto, copy, analytics). Rezultate mai rapide, preturi mai mici.

2. **Preturi transparente, publicate** - Majoritatea agentiilor ascund preturile. Noi le afisam clar. Creeaza incredere.

3. **Dezvoltare + Marketing sub acelasi acoperis** - Client vine pentru site -> ramane pentru marketing (si invers). Nimeni nu ofera asta la preturile noastre.

4. **Fara contracte de minim 6-12 luni** - Abonament lunar, anulezi oricand. IMM-urile urasc lock-in-ul.

5. **Dashboard in timp real** - Nu rapoarte PDF lunare, ci dashboard live unde clientul vede rezultatele 24/7.

6. **Focus pe ROI masurabil** - Nu vandem "awareness", vindem lead-uri, clienti, vanzari.

7. **Specializare pe verticale** - Nu suntem "agentie generala". Ne focusam pe: Beauty/Saloane, HoReCa, Medical/Dental, Fitness.

8. **Support real pe WhatsApp** - Nu chatbot, nu email cu raspuns in 48h. WhatsApp cu raspuns in maxim 2 ore.

### Sloganul

**"Marketing care aduce clienti, nu doar like-uri."**

sau

**"Totul pentru afacerea ta online. De la site la clienti."**

## 12. LEGAL & ADMINISTRATIV

### Structura recomandata

**Varianta A: O singura firma (simplu)**
- Tharsos SRL adauga servicii de marketing la obiectul de activitate
- Pro: Simplu, un singur set de taxe, contabilitate
- Contra: Nu e clar ownership-ul cu partenerul

**Varianta B: Doua firme separate (recomandat)**
- Tharsos SRL = AppRapid.ro (tu, 100%)
- SRL NOU = Agentia de marketing (tu 50% + prieten 50%)
- Contract de colaborare intre cele doua firme
- Pro: Ownership clar, responsabilitati clare
- Contra: Mai multa birocratie, contabilitate dubla

**Varianta C: O firma noua cu ambii parteneri**
- SRL NOU = contine si AppRapid si Agentia
- Pro: Totul sub un acoperis
- Contra: Complicat daca se destrama parteneriatul

**Recomandare: VARIANTA B** - protectie maxima pentru ambii parteneri.
